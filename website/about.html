About the project mdXplorer

The entire concept is to have a remotely-controlled Unmanned Ground Vehicle (UGV) via a web browser. This while using a Raspberry Pi with Arduino and other modules.

The entire assembly consists of a Raspberry Pi 3 B+ which is connected to a mobile hotspot of a smartphone that uses a network carrier SIM with data. The Raspberry will be powered by a portable power bank via MicroUSB.

The Raspberry is providing a way to connect a HQ camera with the built-in camera module port. The camera will be attached to the robot on a tripod, facing forward. To the Raspberry Pi there will be an Arduino Uno board that is powered and communicates with the Raspberry Pi via a MicroUSB cable through Serial Port.

An Arduino camera (ESP32-WROVER-E) with built-in WiFi will be powered from one of the Raspberry’s USB ports. Its WiFi will be connected to the mobile hotspot so that the Raspberry will have access to its live feed through the local network. This camera will be attached on the back of the RPi HQ camera, it facing backwards of the robot.

Both live feeds will be streamed over the internet using the open-source RTSP protocol to a web domain – https://mdxplorer.gymvibeiasi.ro .

The Arduino Uno board will have connected to it the L298N of ST Microelectronics motor driver module that will be powering the four motors, one for each wheel. The drivetrain will be powered by a 9V battery and will receive commands from the Arduino Uno via PWM pins.
